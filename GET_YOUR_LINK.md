## 🎯 Deploy in 3 Steps (2 Minutes)

### Step 1: Install Vercel CLI
```bash
npm install -g vercel
```

### Step 2: Login to Vercel
```bash
vercel login
```
(Opens browser, login with GitHub/Email)

### Step 3: Deploy
```bash
vercel
```

Answer the prompts:
- Set up and deploy? **Y**
- Which scope? **Your account**
- Link to existing project? **N**
- Project name? **career-guidance** (or press Enter)
- Directory? **./** (press Enter)
- Override settings? **N**

**Done! You'll get your link!** 🎉

## 🆘 Troubleshooting

### "Command not found: vercel"
```bash

```

### "Build failed"
```bash
cd frontend
npm install
npm run build
cd ..
vercel
```

### "API not working"
- Check `/api` folder exists
- Check Python files are correct
- Redeploy: `vercel --prod`
