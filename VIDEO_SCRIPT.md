### [1:00 - 2:00] Architecture Overview

**[Show architecture diagram]**

"The system uses a modern, serverless AWS architecture:

At the core, we use AWS Bedrock with Claude 3 Sonnet for dynamic AI conversations. This enables natural language understanding and context-aware responses.

For career ranking, we use an explainable ML engine with transparent scoring - no black-box decisions.

The backend runs on AWS Lambda with API Gateway for RESTful endpoints. We use DynamoDB for session management with 24-hour TTL for privacy.

Career data is stored in S3, and the React frontend is delivered via CloudFront for global performance.

Everything is serverless - zero server management, auto-scaling, and pay-per-use pricing."
